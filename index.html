What Are AI Agent Skills?

AI agent skills are reusable prompts, instructions, and integrations that extend what your AI assistant can do:

  • Claude Code skills — slash commands, hooks, and automated behaviors for the Claude Code CLI
  • Cursor rules — .cursorrules and project-level instructions for the Cursor editor

  • Custom agent tools — MCP server configs, function definitions, and API integrations
