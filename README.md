## What Are AI Agent Skills?

AI agent skills are reusable prompts, instructions, and integrations that extend what your AI assistant can do. Think of them as plug-ins for your AI workflow:

- **Claude Code skills** — slash commands and automated behaviors for Claude Code CLI
- **Cursor rules** — `.cursorrules` and project-level instructions for the Cursor editor
- **Copilot instructions** — custom system prompts and workflow automations for GitHub Copilot
- **Custom agent tools** — function definitions, MCP server configs, and API integrations
